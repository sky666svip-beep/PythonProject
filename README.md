核心功能
1. 单文件去重：支持“查看重复行”或“执行去重处理”。
2. 双文件对比：查找两个文件间的共同行，或根据 A 文件清理 B 文件（差集计算）。
3. 多文件合并：合并两个文件并可选自动去重。
4. 智能引擎：
• 内存引擎：适用于中小文件，速度极快。
• 流式引擎：适用于超大文件（GB级别），占用内存极低。
• 表格引擎：支持 CSV/Excel，可指定特定列（如 ID 列）作为唯一键进行去重。
支持的格式
• 文本类：.txt .log .md
• 表格类：.csv.xlsx, .xls
• 其他类：代码中预留了 .pdf 和 .docx的解析逻辑（通过 parsers/ 模块实现）。
项目结构
main.py: 程序的入口，启动交互式菜单。
• core/: 核心逻辑层，包含 processor.py（调度器）和各类 engine（去重引擎）。
• cli/: 命令行辅助工具，负责 UI 菜单和用户交互。
• parsers/: 负责不同文件格式的读写解析。
