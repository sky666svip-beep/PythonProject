# txt_deduper：多格式文件去重/对比/合并工具
一个轻量、高效的文件处理工具，支持TXT/CSV/Excel等格式文件的**单文件去重**、**双文件对比（交集/差集）**、**多文件合并（可选去重）**，针对不同文件大小（小文件/GB级大文件）设计分层引擎，兼顾性能与内存占用。

## 核心特性
- 📊 多格式支持：适配TXT、CSV、Excel（xlsx）等常见文本/表格文件；
- ⚡ 分层引擎：
  - 内存引擎：小文件快速去重（全量加载，集合/字典去重）；
  - 流式引擎：GB级大文件处理（逐行读写，不占满内存）；
  - 表格引擎：按指定列去重/对比（适配CSV/Excel列结构）；
- 📈 灵活交互：CLI菜单式操作，支持单文件去重、双文件对比（共同行/清理B文件）、多文件合并；
- 🛡️ 容错设计：自动适配文件编码、处理读写异常（文件不存在/权限不足等）。

## 快速开始
### 环境依赖
```bash
# 基础依赖（根据扩展格式补充）
pip install openpyxl  # 处理Excel文件
pip install chardet   # 编码自动检测
pip install tqdm      # 进度条展示（可选）
```

### 使用步骤
1. 克隆/下载项目代码；
2. 运行主程序：
   ```bash
   python main.py
   ```
3. 按照CLI菜单提示选择操作类型（如“单文件去重”），依次输入文件路径、引擎类型等参数，完成文件处理。

## 功能说明
| 功能         | 适用场景                     | 操作说明                                  |
|--------------|------------------------------|-------------------------------------------|
| 单文件去重   | 清理单个文件内的重复行/列    | 选择文件路径 + 引擎类型，输出去重后文件    |
| 双文件对比   | 找两个文件的共同行/清理B文件 | 输入A/B文件路径，选择“共同行”或“清理B（B-A）” |
| 多文件合并   | 合并多个文件并可选去重       | 输入多个文件路径，选择是否去重，输出合并文件 |

## 项目架构
```
txt_deduper/
├── cli/               # 用户交互层：菜单展示、输入接收
├── core/              # 核心调度/引擎层
│   ├── processor.py   # 调度器：根据操作/文件类型分发至对应引擎
│   ├── memory_engine.py  # 内存引擎（小文件）
│   ├── stream_engine.py  # 流式引擎（大文件）
│   └── table_engine.py   # 表格引擎（CSV/Excel）
├── parsers/           # 解析层：适配不同文件格式的读写
├── main.py            # 程序入口
└── README.md          # 项目说明
```

## 扩展与优化
### 格式扩展
可扩展`parsers/`目录下的解析器，支持PDF（PyPDF2）、Word（python-docx）等格式的文本提取与处理。

### 性能优化
- 流式引擎增加分块读写，提升超大文件处理速度；
- 表格引擎加入列索引缓存，减少重复解析耗时。

### 交互增强
支持命令行参数调用（示例）：
```bash
python main.py --dedup test.csv --engine table --column id
```

## 注意事项
1. 处理Excel文件时，确保文件未被其他程序占用；
2. 大文件处理建议选择“流式引擎”，避免内存溢出；
3. 非UTF-8编码文件会自动检测编码，特殊编码可手动指定。
